- Python 3.8 or higher

1.Clone the repository:


2.Navigate to the project directory:

  ```bash
  cd WeatherVerse
  ```

3.Create Virtual environment:

  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  ```

4.Install the required dependencies:
